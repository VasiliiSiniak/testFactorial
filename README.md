## Установка проекта

Выполнить команду в терминале/консоле:

```
npm install
```

## Фреймворк:
- [Playwright (v1.27.1)](https://playwright.dev/)


## Текущие версии
- [Node.js (v18.12.1)](https://nodejs.org/en/)
- [npm (8.19.2)](https://www.npmjs.com/)

## Запуск тестов

- Запуск всех тестов:
```
npx playwright test

```
